# 🌻tinyURL
gin框架编写的短链接服务

使用方法：

```bash
在安装了docker和docker-compose工具之后在docker-compose.yml文件的同级目录下键入如下命令（如果在win或者mac平台下运行，需要将Dockerfile文件中对应go编译参数的部分改变为编译的二进制可以运行在win或者mac平台的编译参数）：
[lomogo@lomogo-kplw0x tinyURL]$ docker-compose up
```
